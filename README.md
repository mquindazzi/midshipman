# midshipman
Midshipman aging project

This research project is related to the research article "Toadily Divergent: Growth and Age Differences Between Two Male Alternative Reproductive Tactics in the Plainfin Midshipman" by Quindazzi et al.

You will note there are various renditions of code that were used for various aspects of the research. If there is any particular confusion as to the stats or data used, please reach out as the code can be muddied in places.
